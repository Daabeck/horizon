# Horizon

Horizon is a minimal static landing page focused on a single visual idea: a mysterious dark horizon with one centered password input.

## Files

- `index.html` contains the stripped-back page structure.
- `styles.css` contains the atmospheric styling and responsive layout.
- `script.js` adds subtle pointer-driven motion.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static file server.
